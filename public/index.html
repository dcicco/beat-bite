      Hello! What you see before you is an application where the user will be able to create, edit and save short music tracks. When first opening up the website, we are greeted with the an animated title on the top of the webpage with the customized logo on the right corner. The user then has some menu options located on the left side of the website. The first option is “saved beats” which will be the menu the user will be able to use when saving the sound bits from the website. The next option will be the location where all the different types of sounds/instruments will be implemented for the user to have to make sound bits. The third option is where the settings will be located. Here you will be able to do things such as change the site theme, change the grid sizing, and other similar features.
